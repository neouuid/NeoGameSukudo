# Neo Sudoku (新·数独)

这是一个使用 Godot 4 开发的现代风格数独游戏。

## ✨ 游戏特性 (Features)

- **经典数独玩法**：自动生成难度适中的数独谜题（基于回溯算法）。
- **现代化 UI 配色**：清新有活力的亮色主题，护眼且美观。
- **自适应窗口**：引入 `AspectRatioContainer`，支持任意调整窗口大小，数独棋盘会完美保持正方形并自适应屏幕剩余空间。
- **便捷操作体验**：专为鼠标和触屏优化的弹出式数字键盘，告别纯键盘输入。
- **实时规则校验**：填写数字时自动检查合法性，冲突数字会用醒目的柔和红色进行提示。
- **清晰的盘面设计**：使用粗边框明显区分 3x3 的大宫格，预设数字与玩家填写数字通过颜色和底色进行区分。

## 🚀 运行方法 (How to Run)

1. 下载并安装 [Godot Engine 4.x](https://godotengine.org/)（推荐 4.1 或更高版本）。
2. 克隆或下载本项目到本地。
3. 打开 Godot 项目管理器，点击 **“导入” (Import)**。
4. 选择本项目目录下的 `project.godot` 文件。
5. 点击 **“运行” (Play)** 按钮（或按 `F5`）即可开始游戏。

## 📁 核心文件结构

- `project.godot`: Godot 项目配置文件。
- `Main.tscn` & `Main.gd`: 游戏的主界面场景及 UI 交互逻辑（包括网格生成、数字键盘弹出、按钮事件等）。
- `SudokuLogic.gd`: 包含数独核心算法的脚本，负责空盘生成、题目挖空以及数字合法性校验。
